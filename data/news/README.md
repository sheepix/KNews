# 新闻数据格式说明

## 文件位置
`data/news/YYYY-MM-DD.json`

## 格式
JSON 数组，每条新闻为一个对象：

```json
[
  {
    "title": "新闻标题（必填）",
    "url": "原文链接（必填）",
    "source": "来源媒体",
    "tag": "分类标签，如：AI前沿 / 国内科技 / 国际科技 / 行业数据",
    "time": "发布时间，如：09:30",
    "summary": "内容摘要"
  }
]
```

## 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| title | ✅ | 新闻标题 |
| url | ✅ | 原文链接，点击跳转 |
| source | ❌ | 来源媒体名称 |
| tag | ❌ | 分类标签，显示为徽章 |
| time | ❌ | 发布时间 |
| summary | ❌ | 摘要内容 |

## 自动生成

你可以通过脚本自动将现有的新闻数据转换为此格式。如果你已经在用 Kimi 的搜索工具收集新闻，可以要求 Kimi 输出为 JSON 格式并存入此目录。
